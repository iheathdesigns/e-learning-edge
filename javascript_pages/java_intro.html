JavaScript is...

A programming language used to modify a web page's structure, content, and style to better interact with the user. JavaScript as a programming language runs in the browser. Most sites presently use JavaScript. In some cases, JavaScript is used to style pages instead of CSS. Users demand more from their browsing experience and JavaScript provides where CSS leaves off.
